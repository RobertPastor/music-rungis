Les fonctions de navigation permettent de parcourir les périodes de l'année. Il existe deux modes de navigation : le mode hebdomadaire et le mode mensuel. Il est possible de passer d'un mode à l'autre par un bouton "va-et-vient" qui affiche toujours l'autre mode disponible. Le bouton de "va-et-vient" est situé en haut et à gauche de la page des réservations. Si le mode d'affichage est hebdomadaire, le bouton "va-et-vient" affiche "mensuel". Si le mode d'affichage est mensuel, le bouton "va-et-vient" affiche "hebdomadaire"
